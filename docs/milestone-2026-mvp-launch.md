## 2. 마일스톤 상세 일정

### M1 — 기능 검증 & OTP 완성 `6/16(월) ~ 6/17(화)` ★ 우선

**목표**: 기존 구현 검증 + 이메일 OTP 완성으로 회원가입 전체 플로우 통과

| 작업 | 담당 | 완료 기준 |
|------|------|---------|
| 이메일 OTP 발송·검증 E2E 테스트 | BE+FE | 실제 이메일 수신 → OTP 입력 → 가입 완료 |
| 카카오 소셜 로그인 E2E 테스트 | FE | Kakao 인증 → 대시보드 이동 → 닉네임 표시 |
| 회원가입 전체 플로우 통합 테스트 | BE+FE | `/signup` → OTP → 약관 → 휴대폰 → 프로필 → 완료 |
| 카카오 알림톡 비즈채널 신청 | 운영 | 신청서 제출 완료 (승인 1~2주 소요) |

> ⚠️ **리스크**: 카카오 알림톡 비즈채널 심사는 외부 의존. 즉시 신청하지 않으면 런치 시 알림톡 발송 불가 → 이메일 폴백으로만 운영해야 함.
